# AwesomeWM on Pop!_OS

## Installation
```bash
sudo apt install awesome wmctrl rofi pnmixer pavucontrol arandr autorandr lxappearance nitrogen
```

### Chrome session
```
# ~/.var/app/com.google.Chrome/config/chrome-flags.conf
--password-store=gnome-libsecret
```

### Mouse cursor size
```
# ~/.Xresources
Xcursor.size: 16
```
